Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem fundamental para criar páginas web. É uma linguagem de marcação que define a estrutura do seu conteúdo web.

Por que aprender HTML?

  • É a base de qualquer página web
  • Fácil de aprender
  • Compatível com todos os navegadores
  • Fundamental para desenvolvimento web

Estrutura Básica

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>

Tags Principais

Tags HTML são os elementos fundamentais que definem a estrutura e o conteúdo de uma página web. Cada tag tem uma função específica e pode conter outras tags ou conteúdo.

Como funcionam as tags?

  • Tags geralmente vêm em pares: tag de abertura e fechamento
  • Formato: <tag>conteúdo</tag>
  • Algumas tags são auto-fecháveis (ex: <img>, <br>)
Tag Descrição
<h1> - <h6> Títulos e subtítulos
<p> Parágrafos
<div> Divisão/seção
<span> Texto inline

Exemplos práticos:

<h1>Título Principal</h1>
<p>Este é um parágrafo.</p>
<div><span>Texto em linha</span></div>

Elementos HTML na Prática

Exemplo Prático

<div class="container">
    <h1>Meu Blog</h1>
    <p>Este é um parágrafo de exemplo.</p>
    <span>Texto em linha</span>
</div>

Resultado:

Meu Blog

Este é um parágrafo de exemplo.

Texto em linha

Atributos HTML

Atributos fornecem informações adicionais aos elementos HTML e sempre são especificados na tag inicial.

Atributos Comuns:

Atributo Descrição Exemplo
class Define uma ou mais classes para estilização <div class="container">
id Define um identificador único <div id="header">
style Define estilos CSS inline <p style="color: blue;">

Listas em HTML

HTML oferece três tipos de listas para organizar seu conteúdo.

1. Lista Não Ordenada

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Resultado:

  • Item 1
  • Item 2
  • Item 3

2. Lista Ordenada

<ol>
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
</ol>

Resultado:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Links HTML

Links são elementos fundamentais que permitem a navegação entre páginas web.

Sintaxe Básica

<a href="https://www.exemplo.com">Visite nossa página</a>

Tipos de Links

  • Link Externo: <a href="https://www.google.com" target="_blank">Google</a>
  • Link Interno: <a href="/contato">Contato</a>
  • Link para Email: <a href="mailto:exemplo@email.com">Envie um email</a>

Imagens em HTML

Imagens são elementos essenciais para tornar seu site mais atraente e informativo.

Sintaxe Básica

<img src="imagem.jpg" alt="Descrição da imagem" width="300" height="200">

Atributos Importantes

  • src: Caminho da imagem
  • alt: Texto alternativo (importante para acessibilidade)
  • width: Largura da imagem
  • height: Altura da imagem

Tabelas HTML

Tabelas são usadas para organizar dados em linhas e colunas.

Estrutura Básica de uma Tabela

<table>
    <thead>
        <tr>
            <th>Cabeçalho 1</th>
            <th>Cabeçalho 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Dado 1</td>
            <td>Dado 2</td>
        </tr>
    </tbody>
</table>

Formulários HTML

Formulários são essenciais para coletar dados dos usuários.

Elementos Comuns de Formulário

<form action="/enviar" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <label for="mensagem">Mensagem:</label>
    <textarea id="mensagem" name="mensagem"></textarea>

    <button type="submit">Enviar</button>
</form>

Tipos de Input

  • text - Texto simples
  • password - Senha
  • email - Endereço de email
  • number - Números
  • checkbox - Caixas de seleção
  • radio - Botões de opção

HTML Semântico

HTML semântico usa tags que transmitem significado sobre seu conteúdo.

Exemplo Prático

<header>
    <h1>Meu Site</h1>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Sobre</a></li>
        </ul>
    </nav>
</header>
<main>
    <article>
        <h2>Título do Artigo</h2>
        <p>Conteúdo do artigo...</p>
    </article>
</main>
<footer>
    <p>© 2025 Meu Site</p>
</footer>

Resultado:

Meu Site

  • Home
  • Sobre

Título do Artigo

Conteúdo do artigo...

© 2025 Meu Site

Tags Semânticas Principais

Tag Uso
<header> Cabeçalho da página ou seção
<nav> Menu de navegação
<main> Conteúdo principal
<article> Conteúdo independente
<section> Seção de conteúdo relacionado
<footer> Rodapé da página ou seção

Vamos criar um CV moderno e responsivo utilizando Bootstrap 5. O Bootstrap foi escolhido por:

Por que usar Bootstrap?

  • Sistema de grid responsivo pronto para uso
  • Componentes pré-estilizados (cards, botões, navegação)
  • Classes utilitárias para espaçamento e alinhamento
  • Design moderno e profissional com mínimo esforço
  • Compatibilidade cross-browser

Dicas de Customização:

  • Personalize as cores usando as classes do Bootstrap (bg-primary, bg-success, etc)
  • Adicione mais seções conforme necessário (Projetos, Idiomas, etc)
  • Use ícones do Bootstrap Icons para melhorar o visual
  • Mantenha o design limpo e profissional
  • Garanta que está responsivo testando em diferentes tamanhos de tela
